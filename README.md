# Адаптивные часы (HTML/CSS/JS)

Готовая веб-страница с локальным временем пользователя:
- цифровой и аналоговый режим (римский циферблат в аналоговом режиме);
- светлая и тёмная тема;
- адаптивная верстка для desktop и mobile;
- без сторонних библиотек.

## Состав проекта

- `index.html` — структура страницы и элементы управления;
- `styles.css` — стили, темы и адаптивность;
- `script.js` — логика часов и переключателей.

## Запуск

1. Откройте папку проекта.
2. Запустите `index.html` в браузере (двойной клик или "Open with Live Server").

Дополнительно:
- Переключения темы и режима запоминаются в `localStorage`.
